## Automation Workflow

### Weekly Update (Every Monday)
1. **Fetch Videos:** `fetch_videos.py` pulls latest carnivore YouTube content
2. **AI Analysis:** `content_analyzer.py` uses Claude to analyze trends
3. **Sentiment:** `add_sentiment.py` analyzes comment sentiment
4. **Generate Pages:** Creates homepage, archive, channels
5. **Newsletter:** AI-generated unique weekly newsletter
6. **Deploy:** Auto-commits ‚Üí triggers GitHub Pages deployment

### Newsletter Distribution
- HTML newsletter saved to Google Drive
- n8n watches folder ‚Üí triggers Mailchimp campaign
- Fully automated delivery
