# Available Skills for Carnivore Weekly

## Brand & Content
- `/carnivore-brand` - Brand standards, design system, voice, and owner preferences
- `/copy-editor` - Copy editing standards to ensure human-quality writing and avoid AI detection
- `/ai-text-humanization` - Transform robotic documentation into natural, engaging human-friendly text
- `/content-integrity` - Validates that automated edits preserve original meaning, sentiment, and facts

## Design & UX
- `/bento-ux-analyzer` - Analyze and improve bento-style website layouts using Playwright
- `/form-optimization` - Expert guidance for creating and auditing web forms to maximize completion rates
- `/html-to-pdf-styling` - Expert guidelines for styling HTML documents for PDF conversion
- `/visual-validator` - Visual and performance validation using Playwright

## Validation & SEO
- `/seo-validator` - Validates SEO requirements for Carnivore Weekly pages before deployment
- `/soft-conversion` - Soft conversion framework for natural product recommendations without pressure tactics

## Operations & Security
- `/security-operations` - Comprehensive security audits, infrastructure operations, compliance, and incident response

---

## How to Use Skills

Invoke any of these skills using the `/skill-name` syntax in Claude Code. For example:

```
/carnivore-brand
/seo-validator
```

Skills provide specialized capabilities and domain knowledge for the Carnivore Weekly project.
