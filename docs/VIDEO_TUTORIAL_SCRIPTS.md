## Production Guidelines

### Video Specifications

**Format:**
- 16:9 aspect ratio (HD or 4K)
- 30fps frame rate
- H.264 codec (MP4)

**Editing Software:**
- Adobe Premiere Pro, DaVinci Resolve, or Final Cut Pro

**Audio:**
- Clear, professional voiceover
- Consistent volume (-14dBFS target)
- Royalty-free background music (optional, very subtle)
- No background noise

**Graphics:**
- Title cards for each video
- Lower thirds with feature names
- Animated icons for features
- Color-coded sentiment bars (green/gray/red)
- Callout boxes for key points

### Voiceover Tips

**Tone:**
- Friendly and approachable (not corporate)
- Clear pronunciation
- Natural pacing (not too fast)
- Conversational (like talking to a friend)

**Pacing:**
- Read script 10% slower than feels natural
- Pause between sentences (1-2 seconds)
- Let visuals speak during visual-heavy sections
- Match voiceover to on-screen action timing

### Recording Screen Footage

**Setup:**
- Use screen recording software (ScreenFlow on Mac, OBS on Windows/Linux)
- Record at full resolution (1920x1080 minimum)
- Use a clean browser (no extensions, minimize notifications)
- Make the window large so text is readable

**Best Practices:**
- Slow down mouse movements for clarity
- Pause 1-2 seconds on each screen before moving
- Enlarge text in browser if needed for readability
- Turn off notifications/Slack while recording
- Test audio input levels before recording

### Post-Production

**Editing:**
- Cut out dead air
- Smooth transitions between clips (0.3-0.5 second fades)
- Add title cards at beginning and end
- Sync audio to visuals precisely
- Color correct to match branding

**Captions:**
- Auto-generate then manually review
- Correct any errors
- Style consistently
- Ensure readability on small screens
- Include keywords for accessibility

### Distribution

**Platforms:**
- YouTube (primary)
- Embedded on carnivoreweekly.com
- Share on social media (TikTok, Instagram Reels 30-sec cuts)

**YouTube Metadata:**
- Title: "[Feature Name] - Carnivore Weekly Tutorial"
- Description: Feature overview + link to full documentation
- Tags: carnivore, diet, tutorial, education
- Thumbnail: Screenshot of key moment + logo
- Playlist: "Carnivore Weekly Tutorials"
