5. Responsive Design

Mobile-first approach with three major breakpoints. Test at 375px, 768px, and 1100px for optimal experience.

Breakpoints

Mobile (375px - 767px)

  • Single-column layout
  • 16px body font (14px mobile)
  • 12px padding
  • 15px grid gap
  • 44px touch targets (minimum)

Tablet (768px - 1099px)

  • 2-column grid layout
  • 15px body font
  • 20px padding
  • 18px grid gap
  • 48px touch targets

Desktop (1100px+)

  • 3-column grid layout
  • 16px body font (standard)
  • 40px padding
  • 20px grid gap
  • Max-width: 1400px container

7. Refined Traditional vs Bento Grid

Evolution from flowing sections to structured grid system while maintaining premium brand aesthetic.

Refined Traditional

  • Flexible, flowing section layouts
  • Varied card heights (content-based)
  • Inconsistent spacing (15-40px ad hoc)
  • H1: 48px, H2: 32px
  • Tan gradient backgrounds
  • Mobile-friendly but no strict system
  • Moderate hover effects

Bento Grid (New)

  • Structured CSS Grid system
  • Standardized card dimensions
  • Strict 20px/30px/40px grid
  • H1: 48px, H2: 32px, H3: 24px (new)
  • Dark brown with tan accents
  • Mobile-first with tested breakpoints
  • 5px lift + scale for featured cards

Key Improvements

  • Visual Consistency: Strict spacing and sizing rules reduce design debt
  • Developer Efficiency: Reusable components vs custom layouts per page
  • Accessibility: Systematic testing across breakpoints and devices
  • Maintenance: Easy to update system-wide through CSS variables
  • Scalability: New pages/components follow established patterns
  • Premium Feel: Dark brown elevates sophistication vs tan-dominant design

Implementation Examples

CSS Custom Properties (Recommended)

:root {   --color-dark-brown: #1a120b;   --color-tan: #d4a574;   --color-gold: #ffd700;   --font-display: 'Playfair Display', Georgia, serif;   --spacing-md: 20px;   --transition-normal: 0.2s ease; }

Card Component CSS

.card {   background: linear-gradient(135deg, #8b4513 0%, #6d3819 100%);   color: var(--color-light-text);   border-left: 6px solid var(--color-tan);   padding: var(--spacing-md);   transition: all var(--transition-normal); }   .card:hover {   transform: translateY(-5px) scale(1.02);   box-shadow: 0 10px 30px rgba(0,0,0,0.6); }
