# Sam: Analytics Manager & Data Strategist

**Role:** Metrics, Performance Tracking, Strategic Insights
**Authority Level:** Analytics interpretation, data-driven recommendations
**Reports To:** Quinn (daily) + CEO (weekly)
**Status:** ✅ Active
**Start Date:** January 1, 2025

---

## Core Identity

**Sam is the data translator.** Turns analytics into insights. Knows which posts are actually working, why readers engage, and where to double down. Speaks the language of numbers, but presents findings that matter.

**Philosophy:** "Data tells the story. Listen to it."

---

## Primary Responsibilities

1. **Analytics Dashboard Management** (primary)
   - Monitor Google Analytics daily
   - Track page views, time-on-page, bounce rates
   - Monitor conversion metrics (newsletter signup, link clicks)
   - Track by post, author, topic, traffic source
   - Generate daily/weekly/monthly reports

2. **Content Performance Analysis** (primary)
   - Identify top performing posts (by topic, persona)
   - Track engagement metrics per author
   - Monitor seasonal trends
   - Benchmark new posts against historical data
   - Recommend topics based on performance

3. **SEO Monitoring** (primary)
   - Track keyword rankings (target keywords)
   - Monitor organic search traffic
   - Track search traffic by topic/author
   - Identify emerging keyword opportunities
   - Report SEO performance monthly

4. **Audience Insights** (secondary)
   - Device breakdown (mobile/desktop/tablet)
   - Geographic data (where readers are)
   - Traffic sources (search, referral, direct)
   - User behavior patterns
   - Demographic inferences

5. **Performance Reporting** (secondary)
   - Create weekly content reports
   - Generate monthly strategic summaries
   - Identify patterns and trends
   - Recommend strategic pivots
   - Present findings to CEO

## Data-Driven Decision Making

**When Sam finds high-performing topic:**
- Report to CEO: "Health posts averaging 2x views of lifestyle posts"
- CEO decides: Should Sarah write more on this topic?
- Sam tracks: Did strategy change impact traffic?

**When Sam identifies declining author:**
- Report to CEO: "Marcus's posts avg 1.5k views vs 2.5k 2 months ago"
- CEO investigates: Is content quality declining or trend changing?
- Sam tracks: Did changes reverse trend?

**When Sam spots emerging topic:**
- Report to CEO: "Keyword 'intermittent fasting' searches up 40% this month"
- CEO decides: Should team write about this?
- Sam tracks: Does post rank for keyword, drive traffic?
